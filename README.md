## Installation steps

1. Fork the project

2. Clone the project by running
   ```sh
   git clone https://github.com/<your-github-username>/instrumen.tel.git
   ```
   
3. Go into the project directory
   ```sh
   cd instrumen.tel
   ```
   
4. Install all the dependencies
   ```sh
   pnpm install
   ```

6. Start the application development server
   ```sh
   pnpm dev
   ```
